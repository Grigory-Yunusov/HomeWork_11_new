

Для створення віртуального середовища та керування пакетами я рекомендую використовувати `pipenv` або `poetry`, два популярних інструменти для роботи з Python-проектами.

1. Якщо ви використовуєте `pipenv`, встановіть його, виконавши команду:

```
pip install pipenv
```

2. Перейдіть у кореневу папку вашого проекту в командному рядку.

3. Щоб створити віртуальне середовище з конкретною версією Python, запустіть наступну команду:

```
pipenv --python <версія Python>
```

Замість `<версія Python>` вкажіть бажану версію, наприклад, `3.9` або `3.8.10`.

4. Зараз у вас створено віртуальне середовище, і файл `Pipfile` автоматично створився для керування пакетами. Ви можете додавати залежності в цей файл вручну або автоматично за допомогою команд `install` або `add`.

```
pipenv install <назва пакету>    # Встановлює пакет
pipenv install --dev <назва пакету>    # Встановлює пакет, який використовується лише підчас розробки
pipenv install    # Встановлює всі залежності, перераховані в Pipfile
```

5. Додайте всі необхідні залежності у файл `Pipfile`, вказуючи їх назву та версію. Наприклад:

```plaintext
[packages]
numpy = "1.21.0"
pandas = "1.3.0"

[dev-packages]
pytest = "^6.2.4"
```

6. Запустіть своє робоче середовище (IDE) та налаштуйте його на використання створеного віртуального середовища. Деталі конфігурації залежать від вашого конкретного робочого середовища.

Ці кроки допоможуть вам створити віртуальне середовище для вашого проекту та налаштувати його залежності.
