Ось поетапна інструкція для створення віртуального середовища для вашого проекту "tech sage" за допомогою `pipenv` та налаштування робочого середовища (IDE) для роботи з ним.

1. **Встановлення `pipenv`:**

   Перш за все, вам потрібно встановити `pipenv`, якщо цього ще не було зроблено. Відкрийте термінал (командний рядок) та виконайте наступну команду:

   ```
   pip install pipenv
   ```

2. **Перехід до папки проекту:**

   Вам потрібно перейти в ту папку, де знаходиться ваш проект. За вмістом папки, який ви надали, команди будуть такими:

   ```
   cd C:\Users\user\Downloads\Home-Work-Python_WEB-1
   ```

3. **Створення віртуального середовища:**

   Після переходу до папки проекту, створіть віртуальне середовище за допомогою `pipenv`, виконавши наступну команду:

   ```
   pipenv --python 3.9
   ```

   В цій команді 3.9 - це версія Python, яка буде використовуватися у вашому віртуальному середовищі. Змініть номер версії, якщо вам потрібна інша версія.

4. **Встановлення залежностей:**

   Якщо у вашому проекті є файл `requirements.txt` із списком залежностей, то за допомогою `pipenv` встановіть їх наступною командою:

   ```
   pipenv install -r requirements.txt
   ```

   Або можна встановити окремі пакети командою `pipenv install <package>`.

5. **Активовання віртуального середовища:**

   Щоб активувати створене віртуальне середовище, використайте команду:

   ```
   pipenv shell
   ```

   Або для запуску команди всередині віртуального середовища без його активації:

   ```
   pipenv run <command>
   ```

6. **Налаштування IDE:**

   Для роботи зі створеним середовищем вам потрібно налаштувати вашу IDE (наприклад, Visual Studio Code, PyCharm тощо):

   - **Visual Studio Code:**
     Відкрийте папку проекту в VS Code, потім натисніть `Ctrl+Shift+P` для відкриття командної палітри та введіть "Python: Select Interpreter". Оберіть інтерпретатор Python, який відповідає віртуальному середовищу, створеному `pipenv`.

   - **PyCharm:**
     Відкрийте налаштування PyCharm (`File > Settings > Project: <назва проекту> > Python Interpreter`), потім оберіть інтерпретатор Python, що відповідає віртуальному середовищу `pipenv`.

Після виконання цих кроків ваш проект буде використовувати віртуальне середовище, створене `pipenv`, із вказаними версіями Python та залежностей.
