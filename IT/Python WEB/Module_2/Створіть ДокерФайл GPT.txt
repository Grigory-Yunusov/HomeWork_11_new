Щоб виконати вашу задачу, спочатку потрібно створити Dockerfile, а потім згенерувати Docker-образ на основі цього файлу. Ось покрокова інструкція, яку ви можете виконати:

Крок 1: Створення Dockerfile
1. Відкрийте текстовий редактор і створіть новий файл з назвою "Dockerfile".
2. Додайте наступний вміст у файл "Dockerfile":

```Dockerfile
FROM python:latest
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
CMD ["python", "tech_sage/main.py"]
```

3. Збережіть файл Dockerfile.

Крок 2: Збірка Docker-образу
1. Відкрийте командний рядок або термінал.
2. Перейдіть в директорію, де знаходиться файл Dockerfile.
3. Виконайте наступну команду, щоб зібрати Docker-образ:

```bash
docker build -t tech-sage-app .
```

Ця команда збирає Docker-образ з назвою "tech-sage-app" на основі Dockerfile в поточній директорії. Зверніть увагу, що крапка на кінці команди важлива.

4. Зачекайте, поки Docker  завершить збірку образу. Після успішного завершення ви побачите повідомлення "Successfully built".

Крок 3: Запуск контейнера з "tech sage помічником"
1. Після успішної збірки Docker-образу, запустіть наступну команду, щоб створити контейнер та запустити "tech sage помічник":

```bash
docker run -t tech-sage-app
```

Ця команда створить контейнер з назвою "tech-sage-container" на основі образу "tech-sage-app" і запустить "tech sage помічник" всередині контейнера.

2. Після виконання команди ви побачите, як "tech sage помічник" починає свою роботу всередині контейнера.

Тепер у вас є окремий контейнер, в якому запущений "tech sage помічник". Ви можете зупинити або видалити контейнер за допомогою відповідних Docker-команд, якщо потрібно.
